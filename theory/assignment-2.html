1: What are variables in JavaScript? How do you declare a variable using var, let, and const? ans = Variables are used to store data values in JavaScript. We can store numbers, text, true/false values, and more in variables. JavaScript has 3 ways to declare variables: 1) var Old way to declare variables. Value can be changed. Can be declared again. 2) let Modern way to declare variables. Value can be changed. Cannot be declared again with same name in same block. 3) const Used for fixed values. Value cannot be changed after declaration. 2: Explain the different data types in JavaScript. Provide examples for each. ans = JavaScript has different types of data: 1) String Used to store text. Example: let name = "Dharmik"; 2) Number Used to store numbers. Example: let age = 20; let price = 99.99; 3) Boolean Stores true or false. Example: let isStudent = true; 4) Null Represents empty value. Example: let data = null; 5) Undefined Variable declared but no value assigned. Example: let city; 6) Object Stores collection of values. Example: let person = { name: "Dharmik", age: 20 }; 7) Array Stores multiple values. Example: let colors = ["Red", "Blue", "Green"]; 3: What is the difference between undefined and null in JavaScript? ans = undefined null - Variable declared but no value assigned - Variable has empty value intentionally - Default value - Assigned by programmer Example of undefined: let name; console.log(name); // undefined Example of null: let name = null; console.log(name); // null
